# FocusFlow

FocusFlow es una aplicación web diseñada para ayudarte a organizar tu tiempo y aumentar tu productividad. Con un calendario intuitivo, gestor de tareas y un planificador especial para estudiar, puedes mantener todo en un solo lugar de forma simple y sin complicaciones.

## ✅ Qué incluye

- **Iniciar sesión segura** - Crea tu cuenta y accede desde cualquier navegador
- **Calendario mensual** - Visualiza tus días de forma clara e interactiva
- **Gestor de tareas** - Añade, edita y elimina tareas con nombre, duración y fecha
- **Planificador de exámenes** - Crea automáticamente tareas de estudio diarias
- **Panel lateral** - Detalles diarios al alcance de un click
- **Tema claro/oscuro** - Adapta la interfaz según prefieras
- **Para móvil y escritorio** - Funciona perfectamente en cualquier dispositivo
